数秘術の概要と特性

数秘術は「命術（めいじゅつ）」に分類される占いです。偶然ではなく、生来持っているデータ（誕生日・名前）に基づき、長期的な運命や性質を分析します。

分類 命術（めいじゅつ）

四柱推命や占星術と同じく、「変えられない情報」を元にします。その人の「設計図」や「取扱説明書」を読み解くのに適しており、瞬間的な吉凶（卜術）とは異なります。

特徴 普遍性とシンプルさ

高度な天文知識や複雑な漢字を必要とせず、万国共通の「数」を用いるため、論理的で現代人にも受け入れやすいのが特徴です。

基本的な計算ロジック

ライフ・パス・ナンバーの出し方

  1. 生年月日を西暦で書き出す (例: 1985年10月24日)
  2. すべての数字を1桁になるまで分解して足す
    1 + 9 + 8 + 5 + 1 + 0 + 2 + 4 = 30
  3. 合計が2桁以上なら、さらに分解して足す
    3 + 0 = 3
  4. 例外ルール: 合計が 11, 22, 33 になった場合は、そこで計算を止める（マスターナンバー）。

※ 流派によって、年・月・日を別々に単数換算してから足す方法もありますが、現代数秘術では全桁加算が一般的です。
