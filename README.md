# Triscope

神秘学と心理学を統合的に探究するためのインタラクティブ・ポータルサイト。

## 🌟 概要

Triscopeは、東洋と西洋の占術・神秘学を体系的に学び、体験するためのWebアプリケーションです。
各占術の歴史的背景、論理的根拠、実践方法をインタラクティブなレポート形式で提供します。

## 🔮 占術カテゴリ

占術は大きく3つのカテゴリに分類されます：

### 命術（めいじゅつ）- Mei

生年月日など不変のデータから、人生の設計図を読み解く占術。

| 占術名 | 説明 |
|--------|------|
| 🪐 西洋占星術 | 天体の配置から魂の設計図を読み解く |
| 🔢 数秘術 | 誕生数や名前の振動から人生のサイクルを分析 |
| ✨ 紫微斗数 | 東洋の星占い、12宮に配置された星々から運命を読み解く |
| ☯️ 四柱推命 | 陰陽五行説に基づく運命学 |
| 🧭 九星気学 | 九つの星と方位から運勢の流れを導く |
| 📜 算命学 | 古代中国の帝王学、人間の本質と天命を算出 |

### 卜術（ぼくじゅつ）- Boku

偶然の中に宇宙の意思を見出し、現在の状況や近未来を読む占術。

| 占術名 | 説明 |
|--------|------|
| 🃏 タロット | 象徴体系を通じて無意識の声を聴く |
| ☯️ 易経 | 陰陽の変化から進むべき道を示す |
| ᚠ ルーン占い | 古代北欧の文字から自然界の力を得る |
| ⛩️ おみくじ | 神仏からのメッセージを読み解く |
| 🔮 水晶球占い | 透明な輝きの中にヴィジョンを受け取る |

### 相術（そうじゅつ）- So

目に見える形から吉凶を判断し、環境を整えて開運する占術。

| 占術名 | 説明 |
|--------|------|
| ✋ 手相 | 手のひらの線から人生の可能性を読み解く |
| 👤 人相 | 顔の造作や表情から性格と運勢を分析 |
| 🏠 風水 | 住環境のエネルギーの流れを整える |
| 🪪 姓名判断 | 名前の画数から運勢を解き明かす |

## 🛠️ 技術スタック

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Styling**: Tailwind CSS (CDN), カスタムCSS
- **Charts**: Chart.js
- **Fonts**: Noto Sans JP, Noto Serif JP (Google Fonts)

## 🚀 使い方

1. `index.html` をブラウザで開く
2. 左メニューまたはメインコンテンツから興味のある占術を選択
3. 各占術のインタラクティブレポートで学習・体験
